📌 Overview

This project implements and analyzes different network switch scheduling algorithms for a 3 × 3 crossbar switch:

Part 1: FIFO Queue (with HoL Blocking)
Part 2: Virtual Output Queue (VOQ – Optimal Matching)
Part 3: iSLIP Scheduling Algorithm
Part 4: Data Visualization (Graphs)

⚙️ Requirements
No installation needed if using Google Colab.

If running locally:
pip install matplotlib

Go to  https://colab.research.google.com

Click File → Upload notebook
